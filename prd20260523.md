# PyZip 产品需求文档 (PRD)

**日期**: 2026-05-23  
**版本**: v0.1  
**状态**: 待完善功能清单

---

## 1. 格式支持

| # | 功能 | 优先级 | 说明 |
|---|------|--------|------|
| 1.1 | RAR 格式读取 | P0 | 支持打开/提取 .rar 归档 |
| 1.2 | 7Z 格式读取 | P0 | 支持打开/提取 .7z 归档 |
| 1.3 | RAR 格式写入 | P1 | 支持创建 .rar 归档 |
| 1.4 | 7Z 格式写入 | P1 | 支持创建 .7z 归档 |
| 1.5 | TAR.BZ2 / TAR.XZ 创建入口 | P1 | 后端已支持，UI 添加对话框缺少对应选项 |
| 1.6 | CAB / ISO / ARJ / LZH 等格式 | P2 | 业界常见格式的读取支持 |

## 2. 压缩核心特性

| # | 功能 | 优先级 | 说明 |
|---|------|--------|------|
| 2.1 | AES 加密/密码保护 | P0 | 创建和提取时支持密码设置 |
| 2.2 | 分卷压缩 (Split to volumes) | P0 | 支持将大归档拆分为多个文件 |
| 2.3 | 自解压 SFX | P1 | 创建可独立执行的解压程序 |
| 2.4 | 压缩级别选择 | P1 | 存储/最快/快速/标准/较好/最好 |
| 2.5 | 固实归档 (Solid archive) | P1 | 提高压缩率，但影响随机访问 |
| 2.6 | 字典大小配置 | P2 | 高级压缩参数调整 |
| 2.7 | 测试归档完整性 (Test) | P0 | 校验归档是否损坏 |
| 2.8 | 修复损坏归档 (Repair) | P1 | 尝试恢复损坏的 .zip/.rar |
| 2.9 | 恢复记录 (Recovery record) | P2 | 为归档添加冗余恢复数据 |

## 3. 用户体验

| # | 功能 | 优先级 | 说明 |
|---|------|--------|------|
| 3.1 | 拖放支持 (Drag & Drop) | P0 | 从系统拖入文件/文件夹，拖出提取 |
| 3.2 | 搜索/筛选文件 | P1 | 在当前目录或归档内按名称搜索 |
| 3.3 | 批量格式转换 (Convert) | P1 | 将归档批量转为其他格式 |
| 3.4 | 归档注释 (Comment) | P2 | 添加/查看归档文字注释 |
| 3.5 | 文件内容预览 | P2 | 在提取前预览归档内文件内容 |
| 3.6 | Shell 右键菜单集成 | P1 | 系统文件管理器右键集成 |
| 3.7 | 设置/偏好对话框 | P1 | 主题、默认路径、压缩参数等设置 |
| 3.8 | 后台操作队列 | P2 | 多个任务排队执行，不阻塞 UI |
| 3.9 | 多语言/国际化 (i18n) | P2 | 支持界面语言切换 |

## 5. 技术债务

| # | 项 | 优先级 | 说明 |
|---|-----|--------|------|

## 优先级定义

- **P1**: 重要功能，应尽快实现
- **P2**: 锦上添花，后期迭代
